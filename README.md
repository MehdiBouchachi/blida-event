# IMALEX Website

Official website for **IMALEX**, an Algerian startup specialized in **natural formulation and scientific R&D** across multiple industries, including cosmetics, nutraceuticals, biopesticides, animal nutrition, and agri-food.

The website is designed to present IMALEX’s vision, expertise, and research-driven approach while serving as a professional entry point for partners, clients, and collaborators.

---

## About IMALEX

IMALEX operates at the intersection of **science, nature, and innovation**.
Its mission is to transform natural resources into **high-performance, validated, and sustainable solutions** through rigorous research and formulation.

Core principles:

- Scientific rigor and evidence-based development
- Sustainable and responsible use of natural resources
- Multidisciplinary R&D approach
- Industry-oriented solutions with real-world impact

---

## Website Objectives

The IMALEX website aims to:

- Present the company’s **vision, values, and positioning**
- Highlight **R&D capabilities** and formulation expertise
- Showcase **application areas** and industries served
- Facilitate **partnerships and collaborations**
- Provide a clear and professional **contact channel**

---

## Industries & Applications

The platform covers IMALEX’s activities in:

- **Cosmetics** – natural and functional formulations
- **Nutraceuticals** – health-oriented, science-backed ingredients
- **Biopesticides** – sustainable crop protection solutions
- **Animal Nutrition** – functional and natural feed innovations
- **Agri-Food** – value-added natural formulations

---

## Tech Stack

The website is built with modern, scalable web technologies:

- **Frontend:** React / Next.js
- **Styling:** Tailwind CSS / CSS Modules
- **Deployment:** Vercel
- **Version Control:** Git & GitHub
